# Urdu Data Set
Contains urdu data Set, training as well as testing for OCR system.

  - This dataSet is converted in binary form, i.e All images are converted to 0 1
  - Orignal images rights belongs to Dr. Hazrat Ali an can be downloaded from <a href = "https://www.kaggle.com/hazrat/uhat-urdu-handwritten-text-dataset">Here</a>
